# VaultRadio

P2P synchronized YouTube radio. One DJ, unlimited listeners. Free forever on GitHub Pages.

## Deploy (same as VaultChat)
1. Create a GitHub repo called `vaultradio` (Public)
2. Upload `index.html` and `manifest.json`
3. Settings → Pages → main → / (root) → Save
4. Share your `username.github.io/vaultradio` link

## How to use
**DJ:** Open app → "Be the DJ" → Enter name → Start → Share 4-digit room code → Add YouTube links to queue
**Listener:** Open app → "Join a Room" → Enter name + room code → Start → Enjoy

## Features
- Synchronized playback — all listeners hear the same thing at the same time
- Queue management — DJ adds/removes tracks, double-click to play any track
- Listeners can suggest songs
- Live chat sidebar
- Volume control per user
- Loop mode
- Works on any phone or PC — just open the link
